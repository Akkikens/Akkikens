## 🍿 What I'm building right now

### [Goldi](https://github.com/climb-together/goldi) — AI Career Coaching Platform @ Climb Together

Goldi is an AI-powered career coaching experience, and I'm the **#1 contributor**. Think of it as a coach that actually listens — with voice mode powered by Hume AI, emotional intelligence built in, and smart interruption handling. We partnered with **Walmart** to build custom onboarding flows targeting different career archetypes (Explorer, Directed, Networker), and shipped a **WhatsApp + SMS** consent & accountability system that drove **+28% re-engagement**. The playlist architecture is URL-driven, multi-playlist, with race condition fixes and navigation caching that just *works*.

> **Stack:** Next.js · Claude AI · Hume AI · WhatsApp/SMS integrations

### [Metta](https://github.com/Metta-AI/metta) — Open-Source Multi-Agent RL Platform @ Softmax

Metta is an open-source research platform for multi-agent reinforcement learning, and I own the **infra & platform layer**. I built a full **Datadog + OpenTelemetry** observability pipeline — metric emission, OTLP receivers, structured collectors that replaced a legacy SQLite setup. On the Kubernetes side, I wrote a watcher service for job lifecycle event processing and storage, plus Helm deploy fixes. Infrastructure-as-code lives in **Terraform** (Lambda, Secrets Manager, IAM for GitHub webhooks). I hardened the CI/CD pipeline with cancelled-run cleanup, failure classification (timeout / OOM / policy_error), and keep-main-green automation. And I set up **Alembic migrations** over PostgreSQL + SQLModel ORM for a 21-table schema.

> **Stack:** Python · Kubernetes · Terraform · AWS · Datadog · OpenTelemetry · PostgreSQL
